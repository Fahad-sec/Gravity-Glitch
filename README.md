# Gravity Glitch

A high-speed, Sci-fi Android arcade game built to challenge gravity and momentum.

## 🚀 Overview
This is my first project built using **Unity** and **C#**. It was developed for learning puposed and for me to get a better understanding of game loops and also to get better at c#.
* **Platform:** Android
* **Performance:** Optimized for a stable **60 FPS**
* **Deployment:** Live on [itch.io](https://solvek007.itch.io/gravity-glitch)

## 🛠️ Tech Stack
* **Engine:** Unity
* **Language:** C#
* **Version Control:** Git & GitHub
* **Build Pipeline:** Signed APK (ARM64) with IL2CPP scripting backend

## Challenges 
- **Unity Engine**: As this was my first project using Unity, I had to learn the engine's features and tools in order to create the game. which was a steep learning curve although after spending some time with it, I did not find it as difficult as I initially thought. 
- **C#**: I wanted to use this project to get hands on experience with C#, which was a new programming language for me. The most difficult part was the strict typing and oop. The OOP part is something I am still getting use to.
